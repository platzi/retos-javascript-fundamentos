# Curso de Fundamentos de JavaScript - Retos

Retos prácticos del Curso de Fundamentos de JavaScript en [Platzi](https://platzi.com).

## Requisitos

- [Node.js](https://nodejs.org/) v18 o superior
- npm

## Instalación

```bash
npm install
```

## Ejecutar tests

```bash
npm test
```
