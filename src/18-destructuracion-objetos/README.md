# Reto 18: Destructuración de objetos en JavaScript

Aprende a dominar la destructuración de objetos, el uso del spread operator y las utilidades de Object.keys, Object.values y Object.entries.

## Ejercicios

| # | Función | Concepto |
|---|---------|----------|
| 1 | `extraerDatosBasicos` | Destructuración básica de propiedades |
| 2 | `extraerNotaRenombrada` | Destructuración con renombrado de variables |
| 3 | `extraerCiudadAnidada` | Destructuración de objetos anidados |
| 4 | `copiarProducto` | Copia superficial con spread operator |
| 5 | `combinarObjetos` | Composición de objetos con spread |
| 6 | `agregarPropiedadesUsuario` | Añadir propiedades inmutablemente |
| 7 | `obtenerClaves` | Object.keys para obtener claves |
| 8 | `obtenerValores` | Object.values para obtener valores |
| 9 | `obtenerEntradas` | Object.entries para pares clave-valor |
| 10 | `objetoAStringArray` | Transformar objeto a array formateado |

## Conceptos clave

- **Destructuración**: Extraer propiedades en variables independientes
- **Renombrado**: Evitar colisiones con `{ propiedad: nuevoNombre }`
- **Spread operator**: Copiar y combinar objetos con `...`
- **Inmutabilidad**: Crear nuevos objetos sin modificar el original
- **Object utilities**: Transformar objetos a estructuras de array
